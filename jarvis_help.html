01Overview

J.A.R.V.I.S (Just A Rather Very Intelligent System) is a desktop AI voice assistant built with Python and CustomTkinter. It runs entirely on your local PC using Ollama (free, no cloud APIs) for AI-powered tasks, with optional AI understanding via llama3.2.

It can control your system, search the web, write content, generate images, answer questions, manage files, and much more—all through natural voice or text commands.

02System Requirements

Minimum

  • Windows 10 or 11 (64-bit)
  • 4 GB RAM
  • Microphone (for voice input)
  • Internet connection (for web search, Wikipedia, translations)

Recommended (for AI features)

  • 8 GB+ RAM
  • Ollama installed with llama3.2 model pulled
  • WebView2 Runtime (for 3D orb, included with Windows 11)

03Setup & Installation

  1. Install Ollama (optional, not needed if bundled in EXE) from ollama.com and pull a model:
    ollama pull llama3.2 ← auto-pulled on first start if bundled
  2. Place nircmd.exe and nircmdc.exe in the same folder as the EXE (for volume control).
  3. Run J.A.R.V.I.S.exe — no installation needed.
  4. Enable AI understanding (auto-enabled if Ollama is running):
    Set JARVIS_AI=0 to disable, or leave unset to auto-enable.

04Getting Started

Voice Mode

Press the MIC button in the bottom-center of the UI and speak your command. The 3D orb pulses when listening.

  • "Hey, open Notepad"
  • "What time is it?"
  • "Search for the latest tech news"
  • "Write a poem about AI"

Text Mode

Type directly in the text box and press Enter. Use Shift+Enter for multi-line input.

Stopping J.A.R.V.I.S

Press the Stop button (red square) next to the text input to cancel the current command or speech.

14NLU Engine (Natural Language Understanding)

J.A.R.V.I.S has a full 5-layer NLU pipeline in nlu_engine.py that analyzes every query.

5 Analysis Layers

LayerWhat It Does
1. LexicalTokenization, POS tagging, lemmatization, sentiment analysis (VADER), text statistics
2. SyntacticPhrase chunking (NP/VP/PP), constituency parsing (CFG), dependency relations
3. SemanticNamed Entity Recognition (person, org, location, date, etc.), Word Sense Disambiguation, hypernym/hyponym, relation extraction
4. DiscoursePronoun resolution ("it", "that" → last topic), topic continuity, follow-up detection, ellipsis resolution
5. PragmaticIntent classification (71 intent types), speech act, politeness/urgency/certainty scoring, question type classification (33 types), negation detection

Detected Intent Categories

greeting farewell thanks praise time_query date_query weather_query definition factoid_query how_to comparison list_query calculate translate spell synonym antonym news_query joke recommendation alarm_set reminder_set email_send note_taking search system_info screenshot volume_control app_launch app_close price_query currency stock_query health_query science_fact technology space_query history_query philosophy recipe_query opinion_query preference help status identity capability

…and 30+ more. J.A.R.V.I.S understands 71 different intent categories.

23Tips & Tricks

  • Natural language: No need to memorize commands — "Hey, lock my PC" works as well as "lock".
  • Follow-ups: Refer to "it" or "that" in your next query. J.A.R.V.I.S remembers the last topic.
  • Text mode: If voice isn't working, use text input. All commands work in text mode too.
  • Stop button: Press the Stop button anytime to cancel speech or a running command.
  • Edit & Rewrite: Click the edit icon on your message to fix a typo, or the rewrite icon to regenerate.
  • Copy responses: Click the copy icon on any message bubble to copy it to clipboard.
  • Chat sessions: Use the sidebar to switch between conversations. History persists across restarts.
  • AI auto-enables: If Ollama is running, AI turns on automatically. Set JARVIS_AI=0 to disable.
  • Screen recording: Use the "screen recording" command to start/stop screen capture.
  • Speed tip: If searches are slow, check your internet connection.
