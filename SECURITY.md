# Security Policy

## Supported Versions

| Version | Supported          |
|---------|-------------------|
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in J.A.R.V.I.S, please report it by
emailing **parwezshahil4@gmail.com**. Do NOT create a public GitHub issue.

You should receive a response within 48 hours. If the issue is confirmed, a
patch will be released as soon as possible.

## Security Features

- **100% Local AI**: All conversations processed locally on your machine
- **No Telemetry**: Zero data collection or usage analytics
- **Offline Capable**: Core features work without internet
- **Environment Config**: Sensitive settings via env vars, never hardcoded
- **Secure Activation**: License key validation with offline fallback
- **Restricted API Access**: `AI_LOCAL_ONLY = True` — only local endpoints allowed
- **Encrypted Storage**: Chat history and notes stored with privacy in mind

## Best Practices

1. Always download from the official website or authorized sources
2. Keep your license key private
3. Review environment variables before setting custom endpoints
4. Run antivirus scans on downloaded installers
