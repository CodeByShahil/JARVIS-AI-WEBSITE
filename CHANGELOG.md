# Changelog

All notable changes to J.A.R.V.I.S AI Assistant will be documented in this file.

## [1.0.0] — 2026-01-15

### Added
- Initial public release
- AI-powered natural conversations with local LLM inference
- 5-layer NLU engine (lexical, syntactic, semantic, discourse, pragmatic)
- Real-time speech recognition and text-to-speech
- System automation (applications, volume, power management, etc.)
- Web search, weather, news, and Wikipedia lookups
- Image generation via API with model fallback
- Chat history with session management
- 3D AI Nexus Orb visualization (WebView2)
- CustomTkinter modern UI with cyberpunk design
- Notes, reminders, alarms, and productivity tools
- Activation/license key system
- Environment variable configuration

### Security
- 100% local AI processing — no cloud dependency
- Zero telemetry or data collection
- API keys restricted to local-only endpoints
